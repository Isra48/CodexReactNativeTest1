# CodexReactNativeTest1

Proyecto de React Native con Expo.

## Requisitos
- Node.js 18 o superior
- npm o yarn
- Expo CLI (`npm install -g expo-cli`) opcional para usar los comandos globales

## Instalación
1. Instala las dependencias:
   ```bash
   npm install
   ```
2. Inicia el proyecto:
   ```bash
   npm run start
   ```
3. Usa la aplicación Expo Go en tu dispositivo o un emulador para escanear el código QR y probar la app.

Los scripts también permiten abrir directamente en plataformas específicas:
- `npm run android`
- `npm run ios`
- `npm run web`
